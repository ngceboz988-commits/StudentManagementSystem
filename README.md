Student Management System (OOP Demonstration)

This is a console-based application written in Java (or C++, depending on implementation) that demonstrates core Object-Oriented Programming (OOP) principles through the management of student records.

It provides basic Create, Read, Update, and Delete (CRUD) functionality for student data.

 Key Features

OOP Principles: Utilizes encapsulation, inheritance, and polymorphism.

Student Data: Allows adding, viewing, modifying, and deleting student records.

Interactive Console: User-friendly menu-driven interface.

 Prerequisites

To compile and run this application, you need:

Java Development Kit (JDK): Version 11 or higher.

A Java IDE (like NetBeans, IntelliJ IDEA, or Eclipse) or a command-line environment.

 Getting Started (Command Line)

1. Clone the Repository

git clone [https://github.com/NgceboMaphumulo/student-management-system.git](https://github.com/NgceboMaphumulo/student-management-system.git)
cd student-management-system


2. Compile the Code

Use the Java compiler (javac) to compile the source file:

javac src/StudentManagementSystem.java


3. Run the Application

Execute the compiled class file:

java -cp src StudentManagementSystem


4. Running within NetBeans

Create a new Java Application project in NetBeans.

Copy the StudentManagementSystem.java file into your project's src folder (or the default package).

Right-click the file and select Run File.

 Structure

The project includes the following main component:

src/StudentManagementSystem.java: Contains the main class and necessary sub-classes for implementing the Student and Enrollment entities.
